Para entrada de dados existem algumas possibilidades, a forma mais simples é com o método prompt(), mas de forma geral a entrada de dados vem atráves de "elementos"" dos formulários em HTML

Para saída de dados existem algumas várias possibilidades:

  • window.alert
  • document.write
  • console.log
  • .innerHTML
  • .innerText

O importante é não se prender a isso no começo, aprender formas simples de entrada e saida, para entender melhor é bom ter um conhecimento básico sobre o DOM.

Aqui vou usar uma forma simples que é usando o prompt() para entrada de dados e o alert() para saída de dados.
